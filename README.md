
Candas
======

WIP proof of concept for low-cost static dataframes.

How to build
------------

You need atleast clang 4.0.0 or gcc 7. 

```
clang++ -std=c++1z src/main.cpp -I./src
```
or
```
g++-7 -std=c++1z src/main.cpp -I./src
```
